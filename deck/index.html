Neo4j Hello World with deck.js

Query

CREATE
  hello = { message : 'Hello, ' },
  world = { message : 'World!' },
  hello -[brave:NEXT { message : 'brave Neo4j ' }]-> world
RETURN hello.message + brave.message + world.message AS message
