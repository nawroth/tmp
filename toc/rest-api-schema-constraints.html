Create uniqueness constraint

Create a uniqueness constraint on a property.

Example request

  • POST http://localhost:7474/db/data/schema/constraint/label_1439351986629_1/uniqueness/
  • Accept: application/json; charset=UTF-8
  • Content-Type: application/json 
{
  "property_keys" : [ "property_1439351986629_1" ]
}

Example response

  • 200: OK
  • Content-Type: application/json; charset=UTF-8 
{
  "label" : "label_1439351986629_1",
  "type" : "UNIQUENESS",
  "property_keys" : [ "property_1439351986629_1" ]
}

Get a specific uniqueness constraint

Get a specific uniqueness constraint for a label and a property.

Example request

  • GET http://localhost:7474/db/data/schema/constraint/label_1439351987368_1/uniqueness/property_1439351987368_1
  • Accept: application/json; charset=UTF-8

Example response

  • 200: OK
  • Content-Type: application/json; charset=UTF-8 
[ {
  "property_keys" : [ "property_1439351987368_1" ],
  "label" : "label_1439351987368_1",
  "type" : "UNIQUENESS"
} ]

Drop uniqueness constraint

Drop uniqueness constraint for a label and a property.

Example request

  • DELETE http://localhost:7474/db/data/schema/constraint/label_1439351986410_1/uniqueness/property_1439351986410_1
  • Accept: application/json; charset=UTF-8

Example response

  • 204: No Content
